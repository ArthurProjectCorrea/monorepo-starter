# ❓ FAQ — Perguntas Frequentes

## Como adicionar um novo pacote compartilhado?

1. Crie uma pasta em `packages/` (ex: `utils/`).
2. Adicione um `package.json` e configure os exports.
3. Importe nos apps/pacotes usando o alias do workspace.

## Como atualizar uma configuração global?

- Edite o arquivo correspondente em `packages/config`.
- Rode `pnpm lint` ou `pnpm build` para validar.
- Se necessário, ajuste os imports/extends nos apps/pacotes.

## Como rodar apenas um app?

- Use `pnpm --filter <app> dev` (ex: `pnpm --filter web dev`).

## Como customizar o pipeline do Turborepo?

- Edite o `turbo.json` na raiz.
- Adicione/remova etapas conforme a necessidade do seu fluxo.

## Como garantir que todos usem as mesmas versões de dependências?

- Sempre instale dependências na raiz com `pnpm add -w <pacote>`.
- Use `pnpm up -r` para atualizar recursivamente.

> Para dúvidas mais específicas, consulte os READMEs de cada pacote/app ou abra uma issue.
